Getting Started

Social - Premium Responsive Admin Template

Internet Explorer & Firefox Issues

Internet Explorer 7 (and lower) and Firefox 3.x are not supported.

Premium Assets

Reselling of this assets is prohibited and the license is non transferable. If you would like to use them as a part of your own (another) product, you have to buy your own license.

Glyphicons PRO

This template includes Glyphicons PRO , an Icons Set value at 59 USD, but totally free for you.

You can find the stylesheets and the font icons files in:

./assets/css/glyphicons/
./assets/fonts/glyphicons/

Basic layout

The sidebar content must be placed before the Topbar content.

<!DOCTYPE html>
<html>
   <head>
      <!-- Head content-->
   </head>
   <body>
      <div class="wrapper">
          <aside class="social-sidebar">
             <!-- Sidebar content-->
          </aside>
          <header>
             <nav class="social-navbar">
                <!-- Topbar content -->
             </nav>
          </header>
          <div class="main">
             <!-- Content -->
          </div>
      </div>
   </body>
</html>
Stylesheets
There two important points to noctice:
  • The admin & frontend designs have their own stylesheet file.
  • Style for the free version of the Bootstrap Glyphicons icon has been separated.

You need the include the core

<link href="./assets/css/social.core.css" rel="stylesheet">

Depending on whether you are coding a page for the frontend or backend (admin) you must choose the appropriate css file:

<!-- Admin -->
<link href="./assets/css/social.admin.css" rel="stylesheet">
<!-- Frontend -->
<link href="./assets/css/social.frontend.css" rel="stylesheet">
